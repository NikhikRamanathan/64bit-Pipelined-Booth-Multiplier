# Methodology

This project evaluates three 64-bit multiplier architectures: Array, Radix-4 Booth, and Dadda.

## RTL Designs

The evaluated RTL files are:

| Architecture | RTL File | Top Module |
|---|---|---|
| Array | designs/array/design.sv | array_multiplier_64bit |
| Radix-4 Booth | designs/booth/design.sv | booth_multiplier_64bit |
| Dadda | designs/dadda/design.sv | dadda_multiplier_64bit |

## Area Flow

Area was generated using Yosys and ABC with the SKY130 HD typical Liberty file:

`sky130_fd_sc_hd__tt_025C_1v80.lib`

The mapped area values are taken from `stat -liberty` in the raw Yosys logs under:

`results/area/raw/`

## Timing Flow

Timing was generated using OpenSTA on the SKY130-mapped Verilog netlists generated by Yosys.

A 10 ns virtual clock was used for all three architectures. Timing results are taken from raw OpenSTA reports under:

`results/timing/raw/`

## Power Flow

Power values are taken from raw power reports stored under:

`results/power/`

## Evidence Policy

The final comparison table reports only values that are traceable to raw evidence files in the repository.
